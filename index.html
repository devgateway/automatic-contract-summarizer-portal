Open‑source Contract Summarization

An experimental tool by Development Gateway that uses modern NLP/LLM techniques to extract and summarize key details from public contracts (names, dates, parties, amounts, contacts, and more) into a structured format. Built for transparency and repeatable analysis.

Architecture

High‑level view of the data flow from document to structured summary.

  1. Ingestion: PDFs/Word docs enter via a watch folder or CLI.
  2. Pre‑processing: Text extraction (PDF/Docx), optional OCR, basic cleaning.
  3. Chunking: Split into sections suitable for LLMs; preserve page refs.
  4. LLM Pass: Prompted extraction/summarization to a standard JSON schema.
  5. Post‑processing: Normalize dates, currency, emails; validate required fields.
  6. Output: Persist JSON + a human‑readable summary (Markdown/HTML/CSV).

Tip: swap the LLM with a local or hosted model depending on resources.

Tech Stack (suggested)

  • Python CLI (Typer/Click)
  • Parsing: pdfminer.six, pypdf, python-docx, optional tesseract-ocr
  • Models: Hugging Face / local (Flan‑T5, Llama‑family) or hosted APIs
  • Validation: Pydantic schema for standardized output
  • Packaging: Dockerfile + Makefile

Adjust to match the actual repository implementation if it differs.

Install

# clone
 git clone https://github.com/devgateway/automatic-contract-summarizer.git
 cd automatic-contract-summarizer

# (optional) create & activate venv
 python -m venv .venv
 source .venv/bin/activate   # Windows: .venv\\Scripts\\activate

# install
 pip install -r requirements.txt

# run (example)
 python -m summarizer --input ./samples --out outputs/

Exact commands may differ; check the README (above) for the authoritative steps.

Docker (example)

# build
 docker build -t devgateway/contract-summarizer .

# run
 docker run --rm -v $PWD/samples:/app/samples -v $PWD/outputs:/app/outputs \
  devgateway/contract-summarizer summarizer --input samples --out outputs

Outputs

  • /outputs/*.json — machine‑readable structured fields
  • /outputs/*.md — human‑readable summary
  • CSV export (optional)
