Architecture

High‑level flow from document ingestion to structured outputs.

  1. Ingestion — PDFs/Word via folder or CLI.
  2. Pre‑processing — text extraction; optional OCR.
  3. Chunking — split to model‑sized passages.
  4. LLM Pass — prompted extraction to JSON schema.
  5. Post‑processing — normalize dates, currency, emails.
  6. Outputs — JSON + Markdown; optional CSV export.

You can swap models (local or hosted) without changing downstream steps.
